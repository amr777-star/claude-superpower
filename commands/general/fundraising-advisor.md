### 2. Fundraising Instruments

**Post-Money SAFE (YC Standard — 2025 market standard for pre-seed/seed):**
- No maturity, no interest, no debt
- Converts at next priced round at lower of: Valuation Cap or Discount
- Post-money means: the cap includes the SAFE amount in the post-money
- Founder dilution is knowable at signing (unlike pre-money SAFEs)
```
SAFE Conversion Shares = Investment / (Cap / Fully Diluted Shares)
If Discount: Conversion Price = Price Per Share x (1 - Discount%)
Investor takes whichever gives more shares
```

**Key SAFE terms to negotiate:**
- Valuation Cap (most important term)
- Discount Rate (typically 10-20%)
- MFN clause (Most Favored Nation — match better future SAFE terms)
- Pro-rata rights (right to participate in next round)

**Convertible Note:**
- Interest: 5-8% (accrues, converts to equity)
- Maturity: 12-24 months
- Conversion: At qualified financing, at cap or discount
- If maturity hits without conversion: negotiate extension vs. repayment

**Priced Round (Series A+):**
Standard terms to understand and negotiate:

| Term | Founder-Friendly | Investor-Friendly |
|------|-----------------|-------------------|
| Liquidation Preference | 1x non-participating | 2x+ participating |
| Anti-Dilution | Broad-based weighted average | Full ratchet |
| Board Seats | Founder majority | Investor majority |
| Protective Provisions | Narrow list | Broad veto rights |
| Pro-Rata Rights | Optional | Mandatory + super pro-rata |
| Drag-Along | High threshold (75%+) | Low threshold (50%+) |
| ROFR | None on common | Full ROFR on all transfers |

### 3. Term Sheet Analysis
When analyzing a term sheet, evaluate:

**Economic Terms:**
- Pre-money valuation → Post-money = Pre-money + Investment
- Price per share = Pre-money / Fully Diluted Shares
- Liquidation preference: 1x, 2x, participating vs non-participating
- Dividends: cumulative vs non-cumulative, rate
- Anti-dilution protection type
- Option pool: size and whether created pre-money (dilutes founders) or post-money

**Control Terms:**
- Board composition (seats and voting)
- Protective provisions (investor veto list)
- Information rights and reporting obligations
- Registration rights (S-1, S-3, piggyback)

**Red Flags:**
- Full ratchet anti-dilution
- Participating preferred with no cap
- Investor board majority before Series C
- Redemption rights (forced buyback)
- Pay-to-play provisions
- Broad protective provisions

### 4. Cap Table Modeling

**Pre/Post Round Calculation:**
```
New Investor Ownership = Investment Amount / Post-Money Valuation
Existing Holder New % = Old % x (Pre-Money / Post-Money)
```

**Option Pool Impact:**
If created pre-money (standard):
```
Effective Pre-Money = Stated Pre-Money - Option Pool Value
Founder Dilution = Option Pool is carved from existing holders only
```

**Full Waterfall Analysis:**
At each exit value ($0 to $1B), calculate proceeds for each stakeholder:
1. Return of liquidation preferences (by seniority)
2. Participation (if participating preferred, up to cap)
3. Conversion analysis: compare preferred return vs. as-converted common
4. Distribute remaining pro-rata to common (including converted preferred)

### 7. Valuation Benchmarking
**Pre-seed:** $2-8M post-money (2025)
**Seed:** $8-20M post-money
**Series A:** $25-80M pre-money (requires $1-3M ARR for SaaS)
**Series B:** $80-250M pre-money (requires $5-15M ARR)

**Revenue Multiple Ranges (2025 SaaS):**
- <40% growth: 5-10x ARR
- 40-80% growth: 10-20x ARR
- >80% growth: 15-30x ARR
- Adjustments: +/- for NRR, gross margin, market size, competitive moat
