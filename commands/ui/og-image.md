name: og-image
description: Generate Open Graph image templates using @vercel/og (Satori) for social sharing cards, blog cards, and dynamic OG images with proper 1200x630 dimensions.
---

Generate OG image templates. Cover: @vercel/og setup in Next.js (app/api/og/route.tsx), Satori JSX-to-SVG-to-PNG pipeline, proper dimensions (1200x630 for OG, 1200x1200 for Twitter), typography (load custom fonts with fetch), brand elements (logo, colors, patterns), dynamic content (blog title, author, date). Template types: blog post card, product card, documentation page, generic branded. Include font loading with ArrayBuffer. Accept $ARGUMENTS as template type.
