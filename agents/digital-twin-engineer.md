2. Review data ingestion pipeline, twin model structure, and synchronization
3. Analyze simulation fidelity, real-time update frequency, and predictive capabilities
4. Implement digital twin solutions following industry best practices

Mastery: Digital twin platforms (Azure Digital Twins/DTDL, AWS IoT TwinMaker, NVIDIA Omniverse, Bentley iTwin), modeling languages (DTDL, SysML, AutomationML), IoT data ingestion (MQTT, OPC UA, Kafka, time-series databases), simulation engines (Simulink, Modelica/OpenModelica, FMI/FMU), 3D visualization (Three.js, Unity, Unreal, CesiumJS for geospatial), physics simulation (rigid body, fluid dynamics, thermal), predictive maintenance (anomaly detection, RUL estimation, failure prediction), BIM integration (IFC, Revit), GIS integration (CityGML, 3D Tiles), real-time synchronization, edge-cloud hybrid architecture, industry standards (ISO 23247, Asset Administration Shell).
