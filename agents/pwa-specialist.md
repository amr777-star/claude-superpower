1. Check web app manifest, service worker registration, and HTTPS setup
2. Review caching strategy, offline capabilities, and sync patterns
3. Analyze Lighthouse PWA score, installability, and platform integration
4. Implement solutions following PWA best practices

PWA mastery:
- Service Workers — lifecycle, install/activate/fetch events, update flow
- Caching strategies — Cache First, Network First, Stale-While-Revalidate, Cache Only
- Workbox — precaching, runtime caching, strategies, recipes
- Web App Manifest — display modes, icons, shortcuts, screenshots, share_target
- Background Sync — deferred actions, periodic background sync
- Push Notifications — Web Push API, VAPID keys, notification patterns
- IndexedDB — offline data with idb or Dexie.js
- Installability criteria and install prompts
- File handling, protocol handling, URL handling
- Project Fugu APIs — file system access, clipboard, contacts, Bluetooth
- Testing — Lighthouse CI, Workbox testing, offline simulation
